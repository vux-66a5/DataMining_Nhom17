# Food Ordering Website
##### Original source : https://github.com/kumarharsh13/Food-Ordering-Website
Một trang web đặt đồ ăn thân thiện
## Tính năng :
- Có 2 vai trò : người mua hàng và admin
- Người mua hàng có thể chọn loại hàng và số lượng mua
- Người mua hàng có thể xem thông tin cá nhân và lịch sử mua.
- Người mua hàng có thể thay đổi địa chỉ và password. 
- Admin có thể: : 
    - Thêm đồ ăn mới vào danh sách
    - Thay đổi giá
    - Xem các đơn hàng đang chờ đặt và xử lí
- Dùng cookie để đăng nhập

## Cách cài đặt và chạy :
- Yêu cầu: Có NodeJS, MySQL, Xampp
- run ```git clone https://github.com/vux-66a5/DataMining_Nhom17.git```
- run ```cd DataMining_Nhom17```
- Import file ```foodorderingwesitedb_final.sql``` vào MySQL qua Xampp
- run ``` npm install ``` trong terminal
- run ``` npm start ``` trong terminal.
